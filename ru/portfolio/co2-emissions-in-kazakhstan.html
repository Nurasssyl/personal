Выбросы CO2 в Казахстане

7.08.2024     R   GIS

В этом проекте я визуализировал распределение выбросов углекислого газа на душу населения в Казахстане за 2023 год. Визуализация была создана с использованием языка программирования R и таких библиотек, как tidyverse, geodata, terra, exactextractr, sf и classInt.


Методы и инструменты

Для создания этой визуализации были использованы следующие инструменты и библиотеки:

- tidyverse: Набор пакетов для манипуляции и визуализации данных.
- geodata: Пакет для доступа и работы с географическими данными.
- terra: Библиотека для анализа пространственных данных.
- exactextractr: Пакет для быстрой выборки статистических данных из растров.
- sf: Пакет для работы с векторными данными.
- classInt: Пакет для выбора интервалов классификации.


Описание визуализации

Карта графически представляет выбросы углекислого газа на душу населения в различных регионах Казахстана. Чем темнее область, тем выше выбросы CO2.


Выводы

Анализ карты позволяет сделать несколько ключевых выводов:
1. Аксуский район в Павлодарской области: Этот район считается самым загрязненным по выбросам CO2.
2. Экибастузский район в Павлодарской области: Соседний район занимает второе место по выбросам CO2.
3. Район Биржан сал в Акмолинской области: Этот район следует за ними по высокому уровню выбросов CO2.
4. Мойынкумский район в Жамбылской области: Этот район также выделяется своим более темным цветом, указывающим на высокие выбросы по сравнению с соседними районами.


Резолюция

По данным ПРООН Казахстан, страна входит в десятку стран мира по интенсивности выбросов углекислого газа. Основная доля выбросов CO2 приходится на промышленный сектор. Примечательно, что 35% потребления электроэнергии в стране приходится на электродвигатели, особенно трансформаторы. Эта работа подчеркивает важность решения проблемы промышленных выбросов и изучения устойчивых энергетических решений.
