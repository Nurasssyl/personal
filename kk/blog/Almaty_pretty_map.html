Алматыны PrettyMaps арқылы визуализациялау

13.08.2024     Python PrettyMaps деректерді визуализациялау деректер ғылымы

Бұл жобада мен PrettyMaps кітапханасын пайдаланып Алматы қаласын визуализацияладым. PrettyMaps OpenStreetMap деректерін қолдану арқылы көрікті карталарды жасауға мүмкіндік береді, оларды жолдар, ғимараттар, саябақтар және тағы басқа қабаттармен реттеуге болады.


Қадамдық түсініктеме

Төменде менің GitHub бетімде табуға болатын жоба кодының бөліктері берілген. Әрбір код бөлігі Алматының көрікті картасын жасау процесін түсіну үшін егжей-тегжейлі түсіндірілген.


1. Қажетті кітапханаларды импорттау

Біріншіден, қажетті Python кітапханаларын импорттауымыз керек. Бұл жерде негізгі кітапхана PrettyMaps, сондай-ақ картаны салу үшін matplotlib пайдаланылады. 

    import prettymaps
    import matplotlib.pyplot as plt

Бұл импорттар PrettyMaps функцияларын пайдалануға және карта көрсету параметрлерін баптауға мүмкіндік береді.


2. Картаның орталығын орнату

Келесі қадам - картаның орталық нүктесін анықтау. Бұл мысалда карта Алматы әкімдігіне бағытталған , оның географиялық ендігі мен бойлығы арқылы белгіленеді. 

      location = (43.2567, 76.9286)

Бұл жол Алматы қаласының орталығын анықтайды. Егер басқа аймаққа назар аударғыңыз келсе, бұл координаттарды кез келген басқа жермен ауыстыра аласыз.


3. Көрікті картаны жасау

Орын анықталғаннан кейін, біз енді PrettyMaps пайдаланып картаны жасай аламыз. Төмендегі код картаны радиуспен көрсетеді, ол қоршаған аумақтың қаншалықты қамтылғанын анықтайды. 

      fig, ax = plt.subplots(figsize=(10, 10))
      prettymaps.plot(
          location,
          ax=ax,
          radius=2000,  # Радиус метрмен
          layers={'perimeter': {'circle': False}},
      )
      plt.show()

Бұл код блогы әкімдікке бағытталған радиусы 2 километр болатын картаны жасайды. Картада ғимараттар, жолдар және жасыл аймақтар сияқты әртүрлі қабаттар көрсетіледі, ол қаланың орналасуын толық қамтиды.


4. Картаны баптау

PrettyMaps көптеген баптау мүмкіндіктерін ұсынады. Сіз қабаттарды, түстерді және стильдерді өз қажеттіліктеріңізге сәйкес реттей аласыз. Мысалы, сіз су қоймалары сияқты қосымша қабаттарды қоса аласыз немесе белгілі бір ерекшеліктерді көрсету үшін түс схемасын реттей аласыз.


Қорытынды

PrettyMaps арқылы қалаларды визуализациялау урбанизацияланған аймақтарды көркемдік және аналитикалық тұрғыдан зерттеуге мүмкіндік береді. Нәтижесінде алынған карталар әдемі ғана емес, сонымен бірге қаланың құрылымы мен орналасуын түсінуге пайдалы мәліметтер береді. Бұл жоба PrettyMaps-ті пайдаланып, әлемнің кез келген қаласын, Алматының тамаша үлгісі ретінде визуализациялауға болатынын көрсетеді.
